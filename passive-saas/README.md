# Passive SaaS Applications

Set-and-forget SaaS tools that generate value through APIs, automation, and self-service usage with minimal ongoing maintenance.

## Applications

### [URL Shortener](./url-shortener/)

A link shortening service with click tracking, device detection, referrer analysis, and an analytics dashboard. Users can create short links with custom aliases and monitor performance.

**Port:** 3000

---

### [Screenshot & PDF API](./screenshot-pdf-api/)

A REST API service for capturing website screenshots and generating PDFs from URLs or HTML content. Includes API key authentication, usage tracking, and rate limiting.

**Port:** 3000

---

### [Status Page Generator](./status-page/)

A SaaS application for uptime monitoring with public status pages. Monitor websites and APIs, track response times, manage incidents, and share branded status pages.

**Port:** 3000

---

### [QR Code Generator](./qr-code-generator/)

A full-featured QR code creation tool with customization options and scan analytics. Supports URLs, text, WiFi credentials, and vCard contacts with export to PNG or SVG.

**Port:** 3000

## Quick Start

```bash
# Pick any app and run:
cd <app-folder>
npm install
npm start
```
