# Active SaaS Applications

High-engagement SaaS products that require active user interaction and provide recurring value through daily use.

## Applications

### [AI Content Generator](./ai-content-generator/)

Generate blog posts, social media content, and professional emails using AI-powered templates. Features user authentication, content history, and multiple content types.

**Port:** 3000

---

### [Invoice & Billing](./invoice-billing/)

A professional invoicing application built for freelancers and small businesses. Create invoices, track payments, manage clients, and generate PDF invoices.

**Port:** 3001

---

### [AI Chatbot Builder](./chatbot-builder/)

A no-code platform for creating website chatbots with customizable conversation flows and embeddable widgets. Includes a visual flow builder, theming, and conversation analytics.

**Port:** 3002

---

### [Appointment Scheduler](./appointment-scheduler/)

A booking and scheduling tool for service businesses. Customers can book appointments online while providers manage their calendar, services, and availability with automatic conflict detection.

**Port:** 3004

## Quick Start

```bash
# Pick any app and run:
cd <app-folder>
npm install
npm start
```
